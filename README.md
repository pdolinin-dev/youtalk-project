# YouTalk

Учебный проект по адаптивной верстке сервиса YouTalk.

## Страницы

- `blog.html` - страница блога.
- `article.html` - страница статьи.

## Структура

- `css/` - основные стили проекта.
- `img/` - изображения, иллюстрации, логотипы и иконки.
- `img/articles/` - изображения для карточек блога и страницы статьи.
- `img/icons/` - SVG-иконки интерфейса.
- `js/` - JavaScript для интерактивных элементов. (бургер)

## Соответствие критериям

- Используется семантическая разметка: `header`, `main`, `footer`, `nav`, `article`, `section`, `aside`, `figure`.
- Применяются псевдоэлементы и псевдоклассы: `::before`, `::after`, `:hover`, `:focus-visible`.
- Учтена accessibility: `alt`, `aria-label`, `aria-labelledby`, скрытые подписи для форм.
- Классы оформлены по БЭМ-подходу.
- Реализована адаптивная верстка для desktop, tablet и mobile.
- Иконки используются в SVG, контентные изображения - в PNG/SVG.
- HTML-документы имеют базовую структуру и вложенность.
